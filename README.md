ToDoリスト
=====

このプログラムは新入社員研修の「実習：プログラミング」のサンプルプログラムである。

## ToDo

### アプリの機能

- [x] ToDoリストにタスク入力内容を指定してタスクを追加できる
- [x] 追加直後タスクの完了状態は未完了
- [x] タスク入力内容が空では追加できない
- [x] ToDoリストのタスクの件数を取得できる
- [x] 初期状態ではタスク件数は0件
- [x] タスクを1件追加したら1件
- [x] タスクの完了状態を設定できる
- [x] ToDoリストの完了タスク件数を取得できる
- [x] 指定した位置のタスクを削除できる
- [x] 初期状態ではタスクは登録されていない
- [x] タスクを列挙できる
- [x] 同一タスクの有無を判断できる
- [x] タスクをクリアできるか判断できる
- [x] タスクをクリアできる
- [x] 追加されたタスクの内容を取得できる
- [x] 追加されたタスクの完了状態を取得できる
- [x] タスクを追加したらタスク件数の変更通知が行われる
- [x] タスクを削除したらタスク件数の変更通知が行われる
- [x] タスクをクリアしたらタスク件数の変更通知が行われる
- [x] タスクの完了状態を変更したらタスク完了件数の変更通知が行われる
- [x] タスクを削除したらタスク完了件数の変更通知が行われる
- [x] タスクをクリアしたらタスク完了件数の変更通知が行われる
- [x] タスクを追加したらクリア可否の変更通知が行われる
- [x] タスクを削除したらクリア可否の変更通知が行われる
- [x] タスクをクリアしたらクリア可否の変更通知が行われる
- [x] タスク入力内容の変更通知が行われる
- [x] タスク入力内容を変更したらタスク入力可否の変更通知が行われる

### アプリの見た目

- [x] タイトルバーにタスク件数が表示される
- [x] 追加ボタンを押すと入力したタスクを追加できる
- [x] タスク入力欄でEnterを押すとタスクを追加できる
- [x] 登録したタスクが一覧に表示される
- [x] 入力したタスクと同じタスクが登録されていたら確認メッセージを表示する
    - はいを押したら登録する
    - いいえを押したら中断する
- [x] チェックボックスをクリックするとタスクの完了状態を切り替えられる
- [x] Spaceキーを押すとカーソル行の完了状態を切り替えられる
- [x] 削除ボタンを押すとその行のタスクを削除できる
- [x] Deleteキーを押すとカーソル行のタスクを削除できる
- [x] 起動直後のタスクリストは空である
- [x] クリアボタンを押すとタスクを空にできる
- [ ] タスクが登録されていなければエラーメッセージを表示して中断する
- [x] 画面サイズの変更に追従して画面項目も拡縮される
- [x] 起動直後は画面の中央に表示される
